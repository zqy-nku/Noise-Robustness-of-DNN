# Noise Robustness of Deep Neural Networks

## Dependencies
1. fire
2. torch
3. torchvision
4. PIL
5. numpy
6. openCV
7. matplotlib
8. mxnet
9. gluoncv

## Running Example
1. running self-trained model
```
python3 classify.py test xxx.jpg 
```
2. running pre-trained model
```
python3 pretrain.py --model cifar_resnet110_v1 --input-pic xxx.jpg
```
